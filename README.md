# Cryptocurrencies

## Overview
The purpose of this project is to create a report that includes what cryptocurrencies are on the trading market and how the cryptocurrencies can be grouped together to create a classification system for a new investment portfolio. The initial data will be preprocessed for PCA, then PCA will be used to reduce the data dimensions, the data will then be clustered using K-means, and finally the data will be visualized.
